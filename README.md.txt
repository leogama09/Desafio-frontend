Fiz todo o desafio no react.js

Dividi a parte de desktop e de mobile em duas pastas.
- Pasta "challenge" para parte desktop
- Pasta "challenge-mobile" para parte mobile

É necessário abrir cada pasta no terminal integrado do vscode e digitar "npm start" para poder visualizar.

A parte desktop vai abrir no "http://localhost:9004/" conforme solicitado.

A parte mobile vai abrir numa porta diferente "http://localhost:9005/", para não dar conflito de portas.

PS: Eu dividi cada parte (desktop/mobile) em componentes diferentes dentro das suas respectivas pastas para poder apresentar o projeto mais facilmente.

PS2: Na parte mobile existem 2 sliders.
